Aim:

1. To plot the input and output static characteristic.

2. To calculate the input dynamic resistance from the input characteristics and out dynamic resistance and current gain from the output characteristics of the given transistor.

Apparatus:

An n-p-n transistor BC 107, Resistors 39 kilo ohm and 1 kilo ohm, Two high resistance rheostats, one 0-3V voltmeter, one 0-30 voltmeter, one 0 - 50 micro Ammeter, one 0 - 50 milli Ammeter, two one way keys and connecting wires.

Theory:

In common emitter configuration the emitter is common to both input and output. For noremal operation the Base-Emitter junction is forward biased and base-collector junction is reverse biase. The input characteristics are plotted between IB and VBE keeping the voltage VCE constant. This characteristics is very similar to that of a forward biased diode. The input dynamic resistance is calculated using the formula.

\( r_i = \frac {\bigtriangleup V_{BE}}{\bigtriangleup I_{B}} \, at \, constant \, V_{CE}\)

The output characteristics are plotted between IC and VCCE keeping IB constant. These cureves are almost horizontal. The output dynamic resistance is given by,

\( r_0 = \frac {\bigtriangleup V_{CE}}{\bigtriangleup I_{C}} \, at \, constant \, I_{B}\)

At a given operating point, we define DC and AC current gains (beta) as follows:

\( DC\, current\, gain\, \beta_{dc} = \frac { I_{C}}{ I_{B}} \, at \, constant \, V_{CE}\)

\( AC\, current\, gain\, \beta_{ac} = \frac {\bigtriangleup I_{C}}{\bigtriangleup V_{CE}} \, at \, constant \, V_{CE}\)

Procedure:

a. Input characteristics:
  1. Connect the Circuit as shown in figure.
  2. Keep the voltage VCE as constant at 2 V by varying VCE
  3. Very the input voltage, VBB in steps of 1 V up to 10 V
  4. Measure the voltage, VBE from voltmeter and current, IB through the ammeter for different values of input voltages.
  5. VCE values of 5 V and 10 V
  6. Draw input static characteristics of tabulated values.
  7. At suitable operating point, calculated input dynamic resistance.
b. Output characteristics:
  1. Fix input base current IB at constant value say at 10 microA
  2. Vary the output voltage, VCC in steps of 1V from 0V up to 10V.
  3. Measure the voltage, VCC from voltmeter and current IC through the ammeter for different values.
  4. Repeat above step 2 and 3 for various values of IB = 20 and 30 micro A
  5. Draw output static characteristics for tabulated values.
